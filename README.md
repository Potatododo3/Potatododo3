### projects

**[CPS.CX](https://github.com/Potatododo3/CPS.CX)**
PWA position size calculator for crypto traders. Supports Saltwayer, Neil, Sherlock, and custom signal types.
`JavaScript` `React` `Vite`

**[Flock](https://github.com/Potatododo3/Flock)**
Windows folder-locking GUI app built with customtkinter and packaged via PyInstaller.
`Python` `customtkinter` `PyInstaller`

**[AutoTradingBYBIT](https://github.com/Potatododo3/AutoTradingBYBIT) / [AutoTradingBITUNIX](https://github.com/Potatododo3/AutoTradingBITUNIX)**
Automated trading bots for Bybit and Bitunix. Monitors positions, executes orders, and handles signals.
`Python` `REST API` `WebSocket`

**[CandleCloseBot](https://github.com/Potatododo3/CandleCloseBot)**
Bybit perpetual futures bot that fires alerts on candle close events using APScheduler.
`Python` `APScheduler` `Bybit API`

**[telegramforwarder](https://github.com/Potatododo3/telegramforwarder)**
Forwards messages from Discord channels to Telegram in real time.
`Python` `Discord API` `Telegram Bot`

**[study-notes-bot](https://github.com/Potatododo3/study-notes-bot)**
Telegram bot that organizes study notes using Forum Topics as folder-like storage.
`Python` `Telegram Bot API`

**[CompassLocator](https://github.com/Potatododo3/CompassLocator)**
Minecraft plugin that lets players track targets on a compass via /settarget.
`Java` `Spigot` `GPL-3.0`
